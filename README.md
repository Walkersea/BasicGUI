## 🚀 Installation

1. Open your preferred Roblox executor (e.g. Synapse X, KRNL, Fluxus, etc.)
2. Copy the contents of `Byte_EXPLOITS_v2.lua`
3. Paste into your executor's script box
4. Click **Execute**

> ⚠️ Make sure your executor has `HttpService` and `CoreGui` access enabled.

## 🧭 Usage Guide

Once executed, the GUI will animate open in the top-left corner of your screen. Click any **tab** at the top of the panel to switch between feature categories. Toggle switches click to enable/disable. Buttons execute instantly on click.

To move the window — **click and drag** the title bar.

## 📑 Tabs Overview

### ⚡ Player
| Feature | Type | Description |
|---|---|---|
| Fly | Toggle | Camera-direction fly using WASD + Space/Shift |
| Fly Speed | +/− Control | Adjust fly speed from 10 to 500 |
| Infinite Jump | Toggle | Jump again mid-air repeatedly |
| Noclip | Toggle | Walk through walls and parts |
| Anti-AFK | Toggle | Prevents automatic idle kick |
| WalkSpeed | +/− Control | Adjust character walk speed (0–500) |
| Reset WalkSpeed | Button | Resets walk speed to default (16) |

### 🌍 World
| Feature | Type | Description |
|---|---|---|
| Boost FPS | Button | Strips particles, shadows, reflections for max FPS |
| Fullbright | Button | Maxes ambient lighting, removes fog and effects |
| Time of Day | Buttons | Set time to Dawn / Noon / Sunset / Night |
| Remove Fog | Button | Sets fog to an unreachable distance |
| Dense Fog | Button | Applies thick grey fog to the world |

### 🛠 Tools
| Feature | Type | Description |
|---|---|---|
| Give Sword Tool | Button | Spawns a sword tool in your backpack |
| Give BTools (All 4) | Button | Gives all 4 standard building HopperBins |
| Give Grab Tool | Button | Gives a game grab tool HopperBin |
| Respawn Character | Button | Force-respawns your character |
| Teleport to Spawn | Button | Moves you to the map's SpawnLocation |
| Teleport to 0,0,0 | Button | Moves you to world origin |

### 🌐 Network
| Feature | Type | Description |
|---|---|---|
| Join Player Server | Button | Finds and teleports you to a specific player's server |
| Server Info | Display | Shows current Place ID, Job ID, and player count |
